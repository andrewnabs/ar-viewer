# Tracking targets

Place the compiled MindAR target here as `scenario1.mind`.

See the "Compile the tracking target" section of `../README.md` for the 4-step process
(drag the radar image into the MindAR web compiler, download `targets.mind`, save it here
renamed to `scenario1.mind`).

Until this file exists, "Start AR" will fall back to the trackless
"Place it in front of me" mode automatically.
